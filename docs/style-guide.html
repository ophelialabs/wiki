Style Guide

This page outlines the style guide for writing articles in the wiki.

Headings

Use headings to structure your articles. Start with an <h1> for the main title, followed by <h2> for major sections, and <h3> for subsections.

Code Blocks

Use the <pre><code> tags for code blocks. For example:


function helloWorld() {
    console.log("Hello, world!");
}
