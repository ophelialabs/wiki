Enterprise Linux

Description

This shell script automatically generates an advanced .NET enterpriselinux project structure with the following features:

  • Creates a standardized solution and project layout
  • Sets up authentication and authorization modules
  • Configures database integration (SQL Server, PostgreSQL)
  • Includes deployment and containerization options
  • Generates necessary configuration files and environment settings
  • Prepares sample microservices and web frontend

Important Directories

  • /, /home, /tmp, /var, /var/tmp, /var/log, /var/log/audit, /boot
  • Always shred or clean drives and motherboards upon disposal to prevent credential harvesting.
  • Using Yum repository for backwards compatibility

Network and Development Tools

  • java, javac, node.js, php, lynx

Cockpit

Cockpit supports Ubuntu Focal (20.04), ROS-Noetic, and Gazebo (Gazebo not supported on RHEL9).

systemctl enable --now cockpit.socket
sudo dnf install -y cockpit-machines

VPN Config Guide

  • Cockpit supports WireGuard VPN (requires FIPS disabled). For production, use Libreswan (Openswan IPsec fork).
